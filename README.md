# AltSpace App

一个简单的Windows应用程序，实现应用搜索功能。

## 功能

- 搜索已安装的应用（从Start Menu加载）
- 实时搜索过滤
- 键盘导航：Enter启动应用，Escape关闭

## 构建和运行

1. 确保安装了.NET 8.0 SDK
2. 运行 `dotnet build`
3. 运行 `dotnet run` 或直接运行 `bin\Debug\net8.0-windows\AltSpaceApp.exe`

应用程序会显示一个搜索窗口，你可以输入应用名称来搜索和启动应用。

## 注意

这是一个简化版本，包含全局热键和系统托盘功能。窗口会直接显示，可以手动测试搜索功能。
